
**********
創建一個單元
**********

   1. 在 **課程大綱** 上，點選您想要創建單元的小節。

   2. 點擊 **建立新單元** 如下的畫面會顯示。

    .. image:: Images/image051.png
       :width: 800


   3. 在 **顯示名稱** 欄位中，填入單元名稱。在 Edge 上畫面上方這個名稱會顯示在單元導覽列中。

      每個單元有一個或多個組件。

     - 創建一個討論空間，您或您的學生就可以發佈問題或者參與討論，點擊 **討論**

     - 創建一個組件，您就可以增加文字、圖片，或者其他內容，點擊 **HTML**

     - 創建一個問題給您的學生解決，點擊 **問題**

     - 新增一個影片，點擊 **影片**
